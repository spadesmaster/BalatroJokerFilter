# Joker Filter

Joker Filter adds compact filter pills directly to **Cartomancer's** Joker toolbar so you can quickly isolate useful subsets of Jokers in very large modded runs.

It is built for practical run management when the Joker row gets crowded and you want to answer questions like:

- Which Jokers are normal slot users and likely replaceable?
- Which Jokers are Negative?
- Which Jokers are temporary, destructive, or sell-based?
- Which Jokers are retrigger engines?
- Which rarity bucket do I want to inspect next?

## Highlights

- Integrated directly into Cartomancer's existing `Hide / Zoom / slider` controls
- Live counts shown on each pill
- Active filter shown in red, inactive pills in blue
- Rarity pill cycles from the rarest currently relevant bucket downward
- Per-button config toggles so users can hide pills they do not want

## Requires

- Steamodded
- Cartomancer
